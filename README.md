## 🌟 Project Overview  

**CogniNote AI** is built for the 2026 information age. It doesn't just store text; it understands it. By integrating a high-speed AI inference engine, the application automatically summarizes long-form content and generates relevant hashtags the moment a note is saved, enabling a smarter workflow for students and professionals.

---

## ✨ Key Features  

- 🤖 **Instant AI Synthesis** – Automated 1-sentence summarization using **Llama 3.1**.  
- 🏷️ **Smart Tagging** – AI-driven hashtag generation for effortless categorization.  
- ⚡ **Ultra-Fast Inference** – Powered by **Groq**, achieving sub-500ms AI response times.  
- 🎨 **Modern Glassmorphism UI** – A sleek, dark-mode interface built with **Tailwind CSS v4**.  
- 📱 **Fully Responsive** – Optimized for seamless use across mobile, tablet, and desktop.  
- 🛠️ **Full CRUD Capabilities** – Create, Read, and Delete notes with real-time UI updates.

## 🖥️ Modules  

### 🔹 AI Intelligence Layer (Groq + Llama 3.1)
- Utilizes the **Llama 3.1 8B** model for efficient, high-accuracy summarization.
- Implements **JSON Mode** to ensure structured data exchange between AI and Backend.
- Features a robust fallback mechanism to handle API rate limits or connectivity issues.

### 🔹 Modern Frontend (Vite 8 + Tailwind v4)
- Uses the latest **Vite 8 "Rolldown"** engine for lightning-fast development builds.
- Styled with **Tailwind CSS v4**, utilizing the new CSS-first configuration and Rust-based engine.
- Includes micro-interactions and loading states for a premium User Experience (UX).

### 🔹 Backend API (Node.js + Express)
- RESTful architecture with clean separation of concerns (Models, Controllers, Routes).
- Secure environment management via `.env` to protect sensitive AI credentials.
- Integrated **CORS** handling for seamless cross-origin communication.
